# Holy Faith School Website

This project is a school website built with React, Vite, TanStack Router, and Supabase-compatible data handling for admissions and contact enquiries.

## Features
- Public school website pages
- Admission enquiry form
- Contact form
- Staff-only admin dashboard
- Teacher management list

## Local development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown in the terminal.

## Build

```bash
npm run build
```

## Deployment
You can publish this repository to GitHub and deploy it using any hosting service that supports Vite/TanStack apps.
